Code Examples

Before (Silent Failure):

generateNewSystem() { const system = createNode(NodeTypes.System); system.generate(); // If this fails, nothing happens window.treeView.addRootNode(system); }

After (With Error Handling):

generateNewSystem() { try { const system = createNode(NodeTypes.System); system.generate(); window.treeView.addRootNode(system); } catch (error) { window.errorHandler.handle('Generate System', error); // User sees dialog, developer sees console log } }

Key Features

  • ‚úÖ Global error catching - No uncaught errors slip through
  • ‚úÖ Native OS dialogs - Professional, platform-appropriate error messages
  • ‚úÖ Console logging - Full error details for debugging
  • ‚úÖ Context awareness - Clear messages about what operation failed
  • ‚úÖ Development mode - Stack traces and DevTools auto-open
  • ‚úÖ Production mode - User-friendly messages without technical details

Error Handler API

Method: handle(context, error)

// Use when you want to show error but not re-throw window.errorHandler.handle('Save Workspace', error);

Method: wrap(fn, context)

// Wrap event handlers const handler = window.errorHandler.wrap(() => { // Your code here }, 'Button Click'); button.addEventListener('click', handler);

Method: wrapAsync(fn, context)

// Wrap async functions const saveAsync = window.errorHandler.wrapAsync(async () => { await saveToFile(path); }, 'Save Operation'); await saveAsync();
